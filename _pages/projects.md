- [ned](https://github.com/caseycronyn/ned) adds modern code editor features to the classic UNIX text editor ed(1) including language server protocol powered code completion, semantically aware syntax highlighting, and line editing interaction. Written in C.
- [Canal chase game](https://uob-comsm0166.github.io/2025-group-14/) set in the pastoral English canal networks. Hosted on GitHub and written in JavaScript, utilising p5.js, the game engine p5play and developed in a team utilising agile methodologies and XP.
- [SQL Database Server](https://github.com/caseycronyn/database) implemented in Java. It mirrors the basic features of a query language, allowing the user to manipulate and retrieve data. Data is stored persistently using the local file system, the program is run with Mvn.
